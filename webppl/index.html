My WebPPL page

WebPPL is a Probabilistic Programming Language system and is available at http://webppl.org/ as an interactive system. The code is available via GitHub (https://github.com/probmods/webppl). However, it's a good idea to install it via node.js (npm install -g webppl) to be able to use it locally, especially since it is often faster (and IMHO neater) to run the models locally than via the web browser.
Unfortunately, WebPPL hasn't been updated for a while and to make it work properly requires a downgrade of the node.js version. Here's what I do when running WebPPL at command line: 
$ nvm use v10.22.0

Some of the packages that are available when running via a browser is not automatically available via the standalone version, e.g. the viz functions, the nice graph functions as well as the physics engine, etc. However, some of them can easily be installed via npm, e.g. the viz, see github.com/probmods/webppl-viz/blob/master/README.md for details.

My WebPPL models

My WebPPL model are often ports from (or via) my BLOG or PSI models. Inside the parenthesis, the "original" source are given. The models are also available at GitHub: github.com/hakank/hakank/tree/master/webppl

Note that some of the models use my WebPPL package hakank_utils (in the directory node_modules/hakank_utils). It must then be run with the --require parameter: 
  $ webppl model.wppl --require hakank_utils
The package includes the following utilities:
  • In hakank_utils.wppl
    • exp_map: Show expectation, MAP, marginals
    • percentile: Percentiles for chain
    • show_stats1: Show stats for a chain
    • show_stats: Show stats for a chain
    • num_bins: number of bins (for histogram)
    • histogram: show histogram for a chain
    • histogram1 (alias to the JavaScript function histogram1, see below)
    • get_samples: get samples for a variable in a chain
    • credibleInterval: credible interval for a variable
    • showCredibleInterval: wrapper for credibleInterval
    • simplex: simplex of an array (ensure that the values sums to 1)
    • draw_without_replacement: draw values from an array without replacement
  • In hakank_utils.js:
    • histogram1
    • pluck
  • In distributions.wppl: Collection of the probability distribution in the *_dist.wppl programs.
